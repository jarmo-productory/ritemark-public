# Contributing to Ritemark

Thank you for your interest in Ritemark!

## Reporting Bugs

Found a bug? Please [open an issue](https://github.com/jarmo-productory/ritemark-public/issues/new?template=bug_report.md) with:

1. **Description** — What happened?
2. **Steps to reproduce** — How can we see the bug?
3. **Expected behavior** — What should have happened?
4. **Screenshots** — If applicable
5. **Environment** — macOS version, Ritemark version

## Feature Requests

Have an idea? [Open a feature request](https://github.com/jarmo-productory/ritemark-public/issues/new?template=feature_request.md) with:

1. **Problem** — What problem does this solve?
2. **Solution** — What would you like to see?
3. **Alternatives** — Have you considered other approaches?

## Code Contributions

Ritemark's source code is currently maintained privately. If you're interested in contributing code, please reach out via an issue first to discuss.

## Questions?

- Open an issue for questions about using Ritemark
- Visit [productory.ee](https://productory.ee) for more about the team
